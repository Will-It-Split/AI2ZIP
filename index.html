🚀 What is AI2ZIP?

AI2ZIP is a browser-based developer tool that takes AI-generated code and automatically rebuilds it into a real project structure.

Instead of manually creating hundreds of files and folders, AI2ZIP detects filenames, creates the folders, lets you edit the code, and exports the finished project as a ZIP file.

✨ Features

  • AI response file detection
  • Automatic folder rebuilding
  • Built-in code editor
  • Project saving
  • ZIP exporting
  • Runs completely locally

⚠️ Why Do I Need To Run A Local Server?

AI2ZIP uses modern browser features such as JavaScript modules, file handling, and local project storage.

When you open the file by double-clicking: 

file:///C:/AI2ZIP/index.html

Browsers block some features for security reasons. This can cause errors and prevent AI2ZIP from working correctly.

Running a local server makes your computer act like a small website host. This allows the browser to safely load all AI2ZIP files.

▶️ How To Start AI2ZIP

1. Download the GitHub repository.

2. Open Command Prompt inside the AI2ZIP folder.

3. Run:

python -m http.server 8000

4. Open your browser and go to:

http://localhost:8000/

AI2ZIP will now run correctly.

🔒 Privacy

AI2ZIP does not upload your files anywhere. Everything happens directly on your computer.

  • No account required
  • No backend server
  • No data collection
